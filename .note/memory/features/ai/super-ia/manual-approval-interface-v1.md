# Memory: features/ai/super-ia/manual-approval-interface-v1
Updated: just now

A aba 'Sugestões IA' (✨) no chat do WhatsApp (/whatsapp/chat) centraliza follow-ups gerados autonomamente com o status 'ready'. O fluxo permite que o operador: (1) clique no cabeçalho da sugestão para abrir a conversa do lead no painel principal SEM sair da aba de sugestões, permitindo avaliar o histórico antes de decidir; (2) edite a mensagem sugerida; (3) envie via Evolution API — após o envio, o sistema navega automaticamente para a conversa do lead para confirmar visualmente o envio. A funcionalidade é restrita ao usuário thiago@sonatura.com.br durante a fase piloto. A configuração do Super IA (/super-ia → Configuração) inclui um seletor de 'Etapas Excluídas' que permite escolher quais etapas do funil NÃO devem gerar follow-ups automáticos. O filtro é aplicado na Edge Function 'super-ia-context' (ação 'get_pending_followups') verificando o 'funnel_stage_id' do lead contra a lista 'excluded_stage_ids' salva no JSON 'ai_followup_config' da organização.
